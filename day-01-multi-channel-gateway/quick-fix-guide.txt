=========================================
🚀 OPENCLAW QUICK-FIX CHEAT SHEET
=========================================
Use these commands if you run into trouble. 
Copy and paste them into your terminal one by one.

-----------------------------------------
1. IF THE INTERNET IS BLOCKED (Telegram won't connect)
-----------------------------------------
Why: Sometimes your internet provider blocks Telegram. This tells your 
computer to use Google's "address book" to find it.

Step A (Fix the connection):
sudo sh -c 'echo "nameserver 8.8.8.8" > /etc/resolv.conf'

Step B (Test if it worked):
curl -4 -v https://api.telegram.org
(If you see "Connected", you are good to go!)

-----------------------------------------
2. IF THE APP IS STUCK (Port 18789 is busy)
-----------------------------------------
Why: If you closed OpenClaw but it's still "hidden" in the background, 
this command completely stops it so you can start fresh.

pkill -f openclaw

-----------------------------------------
3. HOW TO START THE AI
-----------------------------------------
Why: This is the main command to wake up the AI "brain."

ollama launch openclaw --model minimax-m2.7:cloud

-----------------------------------------
4. HOW TO ADD WHATSAPP (New!)
-----------------------------------------
Why: To use WhatsApp, you need a small tool called 'wacli'.

Step A (Install the tool):
# (This command downloads and installs the WhatsApp connector)
curl -sSL https://raw.githubusercontent.com/openclaw/wacli/main/install.sh | bash

Step B (Scan the QR Code):
# (This will show a QR code in your terminal. Scan it with your phone!)
wacli login

-----------------------------------------
5. HOW TO CONNECT TELEGRAM OR WHATSAPP
-----------------------------------------
Why: After you start the app, it will give you a "Code." Use this 
command in a NEW terminal window to link your account.

# For Telegram:
openclaw pairing approve telegram [YOUR_CODE_HERE]

# For WhatsApp:
openclaw pairing approve whatsapp [YOUR_CODE_HERE]

(Note: Delete the brackets and put your code there, like: whatsapp 12345)
